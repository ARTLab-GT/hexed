Detailed Description

template<int n_var, int row_size>
class hexed::Row_rw< n_var, row_size >

"row read/write"

Can read/write to/from a row of quadrature points in an element and/or the associated face quadrature points. This class has only static members, so it is basically just a templated namespace. Note that the dimensionality of the element data is determined by the Row_index argument to the member functions, not inferred from n_var (which is arbitrary regardless of dimensionality).
